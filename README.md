<h1 align="center">
  Filme API
</h1>

API para gerenciar filmes (CRUD).

## Tecnologias

- [Spring Boot](https://spring.io/projects/spring-boot)
- [Spring MVC](https://docs.spring.io/spring-framework/reference/web/webmvc.html)
- [Spring Data JPA](https://spring.io/projects/spring-data-jpa)
- [Arquitetura Hexagonal](https://www.santanderopenacademy.com/pt_br/blog/arquitectura-hexagonal.html)
- [Spring Security](https://spring-io.translate.goog/projects/spring-security?_x_tr_sl=en&_x_tr_tl=pt&_x_tr_hl=pt&_x_tr_pto=tc&_x_tr_hist=true)

## Práticas adotadas

- SOLID
- API REST
- Consultas com Spring Data JPA
- Injeção de Dependências
- Tratamento de respostas de erro
- Segurança e validação de usuários
- Desacoplamento com Arquitetura Hexagonal

## Como Executar

- Clonar repositório git
- Construir o projeto:
```
$ ./mvnw clean package
```
- Executar a aplicação:
```
$ java -jar target/capacitacao-filmes-0.0.1-SNAPSHOT.jar
```
